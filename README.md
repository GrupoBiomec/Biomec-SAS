# Biomec SAS — Software de Análise de Sinais

[![Python 3.12](https://img.shields.io/badge/Python-3.12-blue?logo=python&logoColor=white)](https://www.python.org/)
[![Licença GPL-3.0](https://img.shields.io/badge/Licença-GPL--3.0-green)](LICENSE.txt)
[![Tests & Coverage](badges/coverage.svg)](badges/coverage.svg)
[![Documentação](https://img.shields.io/badge/Documentação-GitHub%20Pages-blue?logo=markdown&logoColor=white)](https://GrupoBiomec.github.io/Biomec-SAS/)


Aplicativo desktop de **Análise Biomecânica** desenvolvido em Python e PyQt6 para processamento, tratamento e exibição gráfica de dados de movimento e forças provenientes de arquivos nos formatos **EMT** e **TDF**.

Projeto acadêmico desenvolvido para o [**Grupo Biomec**](https://github.com/biomec-ufrgs) da **Universidade Federal do Rio Grande do Sul (UFRGS)**, sob orientação do Prof. Dr. Jefferson Fagundes Loss.


## Download

Baixe a versão mais recente do executável para Windows (`.exe`) na página de [Releases](https://github.com/GrupoBiomec/Biomec-SAS/releases/latest).

> Não é necessário instalar Python nem dependências, basta baixar e executar.

---

## Funcionalidades Principais

- Importação de arquivos `.TDF` e `.EMT`.
- Geração e personalização de gráficos.
- Processamento e tratamento de sinais.
- Ferramentas de análise matemáticas e estatísticas.
- Automação de ações por meio de scripts configurados pelo usuário.
- Salvamento de projetos e exportação de resultados.
