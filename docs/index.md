# Biomec SAS — Software de Análise de Sinais

O **Biomec SAS** é uma ferramenta gráfica interativa desenvolvida para facilitar o  processamento de sinais biomecânico coletados por pesquisadores da Universidade Federal do Rio Grande do Sul.

O software permite importar formatos proprietários de laboratório ( `.TDF` e `.EMT`), converte os dados estruturalmente para análise vetorial (usando Pandas DataFrame) e oferece uma interface visual para tratamento, manipulação matemática, análise e exportação de dados.

## Intalação do app

Disponível como executável (`.exe`) para Windows. Acesse a [Página de Releases no GitHub](https://github.com/GrupoBiomec/Biomec-SAS/releases/latest).
